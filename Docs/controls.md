# Controls Guide

## Ground / Taxiing
| Key | Action |
|-----|--------|
| W | Throttle up |
| S | Brake / throttle down |
| A | Steer left |
| D | Steer right |

## Takeoff
Hold **W** until speed reaches takeoff speed, the plane lifts off automatically.

## Flight
| Key | Action |
|-----|--------|
| W | Increase speed |
| S | Decrease speed |
| A | Roll left |
| D | Roll right |
| Up Arrow | Pitch up |
| Down Arrow | Pitch down |
| Z | Yaw left |
| X | Yaw right |

## Systems
| Key | Action |
|-----|--------|
| B | Drop balloon at zone |
| C | Switch camera (3rd person / nose) |
| F | Take photo |
| Tab | Open / close photo gallery |
| Esc | Quit game |
